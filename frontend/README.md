# Frontend

HLX Development Studio frontend application built with Vite and React/TypeScript.

## Contents

- **App.tsx, index.tsx** - Main application entry points
- **index.html, index.css** - HTML template and base styles
- **views/** - Major UI components and views
- **components/** - Reusable React components
- **services/** - Frontend service layer
- **utils/** - Utility functions and helpers
- **config/** - Frontend configuration
- **hlx_runtime/** - Runtime type definitions and integration
- **data.ts, codex_data.ts** - Application data
- **types.ts** - TypeScript type definitions
- **package.json, bun.lock** - Dependencies
- **tsconfig.json, vite.config.ts** - Build configuration
- **tailwind.config.js, postcss.config.js** - Styling configuration

## Key Views

- **Dashboard** - Main dashboard view
- **HLXLPlayground** - HLXL language interactive playground
- **ContractExplorer** - Contract browser and viewer
- **HLXVisualizer** - System architecture visualization
- **CoreRuntimeDesigner** - Runtime design interface
- **LatentSpaceDesigner** - Latent space visualization

## Running

```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Build for production
bun run build
```
